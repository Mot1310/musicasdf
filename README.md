# To-Do List Application

## Overview
This is a simple to-do list application that allows users to manage their tasks efficiently. Users can add, edit, and delete tasks, as well as mark them as completed.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Mot1310/musicasdf.git
   ```

2. Navigate to the project directory:
   ```bash
   cd musicasdf
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
